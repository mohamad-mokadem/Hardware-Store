# Hardware Store

Hardware Store is an e-commerce platform developed for CSC443 (Web Programming). The project was designed to simulate an online hardware store where users can browse products, manage orders, and interact with a complete shopping system.

## Features

- Product browsing and category management
- User authentication and account management
- Shopping cart and order processing
- Admin dashboard and product management
- Database-driven inventory system
- Responsive web interface

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Purpose

The project was developed as part of CSC443 (Web Programming) to demonstrate full-stack web development concepts and e-commerce system implementation.
